用AI做了一个 Vampire Crawlers 的卡牌图鉴工具，方便在游戏中随时查阅，分享给大家。



这个工具能干什么？

103 张卡牌、58 颗宝石、163 个解锁项 
每张卡什么效果，哪张卡怎么解锁，怎么合成，宝石效果获得概率都能查得到。支持全局拼音首字母搜索！！！见下面详细介绍（主要是我自己也懒得打字...）。手机端完整适配，方便电脑打游戏懒得切来切去的。懒得看拖到最下面看地址


卡牌详情：

点击任意卡牌可以查看完整信息 — 攻击力、法力消耗、稀有度、效果描述，还有卡牌图片。



筛选系统：

卡牌按类型筛选（攻击/防御/特殊等），宝石按稀有度筛选，解锁项按分类筛选（角色/遗物/奥术等）。



全局搜索功能：

支持中文名称、英文名称搜索，还支持拼音首字母搜索。比如输入 sb 就能搜到「神鞭」，和有关的合成，解锁条件等。当然宝石也可以。
